Customizable Cover Plates for the Lily58 low-profile split keyboard. This project provides the OpenSCAD files to generate the cover plates.

The project is hosted on MakerWorld: [Lily58 Custom Cover Plate](https://makerworld.com/en/models/1998345)

## Compatibility

These cover plates are designed for the Lily58 Choc Low Profile Keyboard. You can get one from [Typeractive](https://typeractive.xyz/pages/build#lily58_choc).

## Customizer Instructions

1.  **Launch the Customizer on MakerWorld**

    - Click on the "Customize" button on the [MakerWorld page](https://makerworld.com/en/models/1998345).
    - Select a cover plate to customize (Left or Right) and click on "Customize" again.

2.  **Customize & Download**

    - Import and position your SVG in **SVG Settings**.
    - In **Export Settings**:
      - Enable **LAY FLAT** to prepare the object for printing.
      - Set the **OBJECT SELECTION** to `Blank Cover Plate`.
      - Generate the model and download the STL.
      - Set the **OBJECT SELECTION** to `Logo`.
      - Generate the model and download the STL.

3.  **Repeat**
    - Repeat steps 1-2 for the other cover plate.
